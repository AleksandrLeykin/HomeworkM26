# HomeworkM26

Задача: реализуйте подсчёт суммы элементов массива несколькими потоками.
o Массив случайных чисел размером N должен разбиваться на M равных частей (M < N), каждая из которых выполняется в отдельном потоке.
o Каждый поток вычисляет сумму элементов своей части.
o По завершении выполнения потоков, вычисляется общая сумма массива.
Входные переменные:
  N — размер массива;
  M — количество частей, на которые будет разбит массив.
Результат: сумма элементов массива.

1. Создаем вектор случайных чисел с N элементами от 1 до 100.
2. засечь время исполнения программы
